# React hooks

Dans ce projet, nous allons poursuivre l'apprentissage des bases de React. Nous allons voir :
  - Comment transitionner d'un somposant classe à un composant fonction tout en maintenant ses fonctionnalités et en modernisant le code
  - Comment utiliser différents React hooks (useState, useContext, useEffect…)
  - Comment optimiser les composants
  - Comment créer des React hooks personnalisés
  - Comment adapter nos tests pour les hooks
